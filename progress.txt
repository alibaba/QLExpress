## Codebase Patterns
- Package structure for new parser: `com.alibaba.qlexpress4.parser.{token,lexer,parser,ast,visitor}`
- ANTLR grammar files in `src/main/antlr4/` contain the complete language specification
- The QLexer.g4 and QLParser.g4 files are the source of truth for all token types and grammar rules
- Keywords include: for, if, else, while, break, continue, return, switch, case, default, function, macro, import, static, new, type keywords (byte, short, int, long, float, double, char, boolean), null, true, false, extends, super, try, catch, finally, throw, then, class, this
- Operators include arithmetic (+, -, *, /, %, ++, --), bitwise (&, |, ^, ~, <<, >>, >>>), logical (!), comparison (>, <, >=, <=, <>), assignment (=, +=, -=, *=, /=, %=, &=, |=, ^=, <<=, >>=, >>>=), and other operators (., .*, ->, ::, ?., *., ?, :)
- Delimiters: (, ), {, }, [, ], ;, ,
